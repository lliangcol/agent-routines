# node-workspace-check Workflow

## 目的

检查 Node workspace metadata、package-manager 命令、lockfile 和常见 validation/release scripts，不安装依赖，也不发布。

## 支持系统

支持 Windows PowerShell 5.1/PowerShell 7+、macOS Bash/PowerShell 7+ 和 Linux Bash/PowerShell 7+。

## 说明

本文件是面向用户的中文 README companion；workflow 行为以脚本和 schema 为准。
